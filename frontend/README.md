# Task Manager

## Description
Application de gestion de tâches avec API backend et interface frontend.

## Technologies
- Node.js / Express
- React
- MongoDB

## Installation

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev

## Fonctionnalités
- Ajouter une tâche
- Voir les tâches
- Supprimer une tâche
- Modifier une tâche

## Choix techniques
- Express pour API simple
- React pour interface dynamique
- MongoDB pour stockage
