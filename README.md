# 📝 Task Manager App

Application full stack de gestion de tâches développée avec React, Node.js et PostgreSQL.

---

## 🚀 Description

Cette application permet de :
- Ajouter des tâches
- Afficher la liste des tâches
- Supprimer des tâches

---

## 🛠️ Technologies utilisées

Frontend :
- React.js
- Bootstrap
- Axios

Backend :
- Node.js
- Express.js
- PostgreSQL
- pg (node-postgres)
