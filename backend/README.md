Backend - Task Manager API

API du projet Task Manager.

Technologies
- Node.js
- Express
- PostgreSQL

Installation
bash
cd backend
npm install

Lancer le serveur
node app.js

Routes API
GET /api/tasks → obtenir toutes les tâches
POST /api/tasks → ajouter une tâche
DELETE /api/tasks/:id → supprimer une tâche

 Port
Serveur lancé sur : http://localhost:5000
